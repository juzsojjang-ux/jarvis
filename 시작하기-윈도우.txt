============================================
  자비스(JARVIS) 윈도우 설치 - 아주 간단 버전
============================================

[ 설치 방법 - 더블클릭 한 번 ]

압축을 푼 "jarvis" 폴더 안에서
   BUILD_JARVIS.bat  를 더블클릭하세요.

이것만 하면 됩니다. 나머지는 전부 자동입니다:
  - Python 3.11 이 없으면 → 동봉된 설치 파일(python-3.11.9-amd64.exe)로
    조용히 자동 설치합니다. (다른 버전이 깔려 있어도 3.11 을 따로 깔아서 씁니다)
  - 필요한 패키지 설치 → JARVIS.exe 빌드까지 알아서 진행합니다.
  - 검은 창이 뜨고 몇 분 걸립니다. 창을 닫지 말고 기다리세요.

끝나면 폴더가 자동으로 열립니다:
       jarvis\dist\JARVIS\JARVIS.exe

이 JARVIS.exe 가 자비스입니다. 더블클릭하면 켜집니다.
(바탕화면에 바로가기를 만들어 쓰세요.)


[ 왜 압축 안에 바로 실행파일(EXE)이 없나요? ]

윈도우 실행파일은 윈도우에서만 만들어집니다.
(맥에서는 윈도우 EXE 를 만들 수 없어서, 자동 빌드기를 넣었습니다.)
처음 한 번만 BUILD_JARVIS.bat 을 돌리면 됩니다.


[ 참고 ]

* 처음 실행 때 "알 수 없는 앱" 경고가 뜨면
  → "추가 정보" → "실행" 누르면 됩니다. (서명 안 한 개인용 앱이라 정상)

* Claude(구독) 두뇌를 쓰려면 윈도우에 claude CLI 를 따로 설치해야 합니다.
  (맥용 번들 CLI 는 윈도우에서 동작하지 않습니다.)

* 음성 선택 / 이름 변경 등은 첫 실행 설정 화면에서 고르면 됩니다.
