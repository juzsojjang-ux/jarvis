# 자비스(JARVIS) 사용설명서  (v4.0.0)

개인용 음성 비서. **맥·윈도우**에서 로컬 음성 인식·합성 + Claude 구독 두뇌로, 아이언맨의
자비스처럼 말로 부려 쓴다. 영어로 답하고 화면에 한국어 자막을 띄운다.

> **v3.0.0 핵심**: 받으면 **설치 없이 바로** 진짜 자비스 영어 음색(Pocket)으로 동작한다 —
> torch·음색 모델이 앱 안에 들어 있어 별도 설치·HF 토큰이 필요 없다.
>
> **v3.0.1 새 기능**: **타자로도** 자비스와 대화(전역 단축키 → 채팅창), 오브 크기 **단축키
> 토글**, 이름 변경 시 **별칭** 지원, 반복되던 오류 음성 차단. (자세히는 11장)
>
> **v4.0.0 새 기능**: **에이전트 기반 강화** — 보안 게이트·플러그인·권한 자가복구. (자세히는 11장)

## 1. 설치 · 실행

### A. 배포 앱으로 (권장 — 무설치)
1. 릴리스에서 받는다: **github.com/juzsojjang-ux/jarvis/releases/latest**
   - 맥: `JARVIS-mac.zip` · 윈도우: `JARVIS-windows.zip`
2. 압축을 푼다.
3. **맥 첫 실행(1회, Gatekeeper 해제)** — 압축 푼 폴더에서 터미널로:
   ```bash
   xattr -dr com.apple.quarantine JARVIS.app
   ```
   그 다음 `JARVIS.app` 더블클릭. (윈도우는 `JARVIS.exe` 더블클릭)
4. 첫 실행이면 브라우저로 **설정 창**이 뜬다 → 두뇌(Claude/Gemini/GPT) 선택 + 로그인 → "시작".
5. 부팅이 끝나면: **"자비스 준비 완료"**. 음성은 곧바로 **Pocket 자비스 음색**(영어).

> 첫 실행 시 **마이크·입력 모니터링** 권한을 물어본다 — 허용해야 웨이크워드/PTT가 동작(6장).
> 미서명 앱이라 재빌드 시 권한이 풀릴 수 있다(드물게). 제거는 9장의 제거기로.

### B. 소스로 (개발/고급)
```bash
cd ~/jarvis
python3.11 -m venv .venv && . .venv/bin/activate && pip install -e ".[dev]"
.venv/bin/python -m jarvis
```
- 소스 실행은 Pocket을 쓰려면 `.venv-pocket`이 필요하다(없으면 edge로 폴백).

## 2. 말 거는 법

세 가지 방법:

1. **웨이크워드** — 그냥 **"자비스"** 라고 부른다. (마이크만 쓰므로 추가 권한 불필요)
   - "자비스"만 부르면 **3초 동안 듣는다** — 그 안에 말을 시작하면 명령으로 받는다.
   - "자비스, 지금 몇 시야?"처럼 한 번에 말해도 된다.
2. **PTT(눌러 말하기)** — **오른쪽 옵션(Alt) 키**를 누른 채 말하고 떼면 처리.
   (입력 모니터링 권한 필요 — 6장)
3. **연속 대화** — 답변 직후 8초간은 "자비스" 없이 이어 말할 수 있다.
4. **타자로 대화** — 전역 단축키 **⌥Space(맥) / Ctrl+Space(윈도우)** 를 누르면 작은 **채팅창**이
   떠오른다. 타이핑하고 Enter로 보내면 자비스가 **글로** 답한다(조용한 상황·긴 입력에 편리).
   답 옆 **🔊**를 누르면 그 답을 음성으로도 들려주고, 위쪽 "음성" 토글을 켜면 이후 답을 자동으로
   읽어준다. **Esc**로 닫는다(대화 초기화). 명령(타이머·맥 조작 등)도 타자로 된다. 단축키는
   설정 창에서 바꿀 수 있다.

말 끊기(바지인): 자비스가 말하는 중에 키를 누르거나 다시 부르면 즉시 멈추고 새 요청을 듣는다.

> **이름 바꾸기**: 설정 창에서 자비스 이름을 바꿀 수 있다. STT가 새 이름을 다르게 알아듣는 일이
> 잦으면 **별칭**(쉼표로 구분)을 함께 넣어 두면 그 발음들로도 깨어난다.

"통역 모드 켜줘" → 한국어로 말하면 영어로, 영어로 말하면 한국어로 통역. "꺼줘"로 복귀.

### 기억
"나 매운 거 못 먹는 거 기억해" — 장기 기억. 대화 중 선호·약속은 자동으로도 저장.

### 일·코딩 시키기 (풀 도구)
두뇌가 bash·파일 읽기/쓰기/편집·검색을 가진다. "스크린샷 폴더로 정리해줘", "이 폴더에
README 만들어줘" 등 실제 작업 수행.
- **파괴적 단계(파일 쓰기·실행)는 실행 전 "~할까요?" 음성 확인** — "네"라야 실행.
- "전권 모드 켜줘" → 그동안은 확인 없이 실행(10분 자동 만료).

### 능동적 자비스 (먼저 말 걸기)
- 아침 첫 잠금 해제 시 브리핑(필요한 것만 짧게 — 매번 기계적 보고는 안 함)
- 배터리 20/10/5% 경고, 충전·완충 알림
- 일정·미리알림 10분 전 사전 알림
- 오랜만 복귀 인사, 타이머 완료 알림
- (대화 중엔 끼어들지 않고 한가할 때만)

### 진단·사용량
"자가진단 / 상태점검", "사용량 알려줘"

### 아이폰 원격
아이폰 단축어로 텍스트 명령 → 자비스가 텍스트로 답장(부재 중, 읽기 전용). 설정: `docs/REMOTE.md`.

## 4. 보이스 (음성 종류) 와 자막

배포 앱(v3.0.0)은 **Pocket 영어 집사 음색이 앱에 내장**돼 받으면 바로 그 음색으로 말한다
(설치·토큰 불필요). 설정 창(또는 메뉴막대 아이콘 → 설정)에서 다른 보이스로 바꿀 수도 있다 —
바꾼 뒤 **재시작**하면 적용.

| 보이스 | 발화 | 자막 |
|---|---|---|
| **Pocket 영어 집사**(기본, `jarvis`) | 영어(진짜 자비스 클론, 앱 내장) | 한국어 자막 |
| 한국어 여성 / 남성 (`female_ko`/`male_ko`) | 한국어 | 없음(중복이라 끔) |
| 자비스 음색 한국어 (`jarvis_ko`) | 한국어 + 자비스 음색(ONNX) | 없음 |
| 미국 남성/여성 (`male_us`/`female_us`) | 영어 | 한국어 자막 |

- **안전망**: Pocket이 어떤 이유로 실패해도 자동으로 **edge → say**로 폴백해 소리는 항상 난다.
- **자막**: 영어로 말할 때만 한국어 자막을 띄운다(이해용). **한국어로 말하는 모드는 자막을 끈다**.
- 자막은 **문장 단위로, 그 문장이 발화되는 동안** 떠서 타이밍이 맞는다.

## 5. 화면(HUD) · 메뉴막대

- **오브**: 자비스가 반응할 때 화면 **중앙에 크게** 떠 음성에 맞춰 움직인다.
  - "크게 띄워" / "작게" 로 중앙↔코너 전환. 또는 **전역 단축키 ⌥O**(기본, 설정에서 변경)로
    크게 ↔ 우측 하단 작게를 **즉시 토글**.
- **메뉴막대 아이콘**: '자비스 실행 중' 표시. 클릭하면 **설정**(보이스·키·두뇌) 열기, **종료**.

## 6. 권한 (중요)

맥은 기능별로 권한이 따로다. **시스템 설정 → 개인정보 보호 및 보안**에서 켠다.

| 기능 | 필요한 권한 | 비고 |
|---|---|---|
| "자비스" 웨이크워드 | **마이크** | 첫 마이크 사용 시 자동 요청 |
| **오른쪽 옵션(PTT) 키** | **입력 모니터링(Input Monitoring)** | 키보드 감시 — 손쉬운 사용 아님! |
| 화면 제어(클릭·입력) | **손쉬운 사용(Accessibility)** | cliclick |
| "화면 봐줘"·"화면에 뭐 있어" | **화면 기록(Screen Recording)** | 첫 사용 시 자동 요청 |

> **PTT 키가 안 되면**: 시스템 설정 → 개인정보 보호 및 보안 → **입력 모니터링**에서
> 자비스(또는 실행한 Python/Terminal)를 켜고 **재시작**. (소스 실행을 권장 — 권한이 안정적)

## 7. 두뇌 (Claude / Gemini / GPT)

- 기본 **Claude 구독 로그인**(API 과금 없음). 첫 설정에서 버튼 한 번으로 로그인.
- "최대 사고 / 깊게 생각해봐"를 붙이면 그 턴만 확장 사고(느리지만 똑똑).
- 대화 맥락·장기 기억 유지. 한도 초과 시 화면·음성으로 알림.

## 8. 문제 해결

| 증상 | 해결 |
|---|---|
| PTT 키가 안 됨 | 입력 모니터링 권한 확인(6장). **소스 실행**이 가장 확실. |
| 음성이 이상/안 나옴 | 보이스가 Pocket인지(설정), 로그 `~/.jarvis/logs/jarvis.log` 확인. 배포 `.app`은 edge-tts 실패 시 자동으로 macOS `say`로 폴백(소리는 항상 남) |
| "자비스" 불러도 무반응 | 마이크 권한, 마이크 입력 장치 확인 |
| 두뇌가 답 안 함 | Claude 로그인 상태("자가진단"으로 점검), 한도 초과 여부 |
| 자비스가 여러 개 뜸 | JARVIS.app 복사본을 하나만 두기(중복 번들이 원인) |
| 로그 보기 | 소스: 터미널 출력 / 배포: `~/.jarvis/logs/jarvis.log` |

## 9. 안전 장치

- **읽기는 자유, 쓰기·실행은 음성 확인**("~할까요?" → "네").
- **화면 조작은 모드 게이트 + 자동 만료**. 오인식 한 마디로는 파괴적 동작 불가.
- 아이폰 **원격은 읽기 전용**(발송·앱 실행·화면 조작 불가).
- "자비스"로 시작하지 않은 발화는 즉시 폐기(저장·로그 안 함).
- `control_mac`의 위험 명령(셸 실행·삭제 등)은 **코드 차원에서 거부**(LLM 선의에만 의존 안 함).

## 10. 제거 (완전 삭제)

미서명 앱은 휴지통에 버려도 **권한(TCC) 흔적이 남는다**. 깨끗이 지우려면 동봉된 제거기를 쓴다:

- **맥**: `자비스-제거.command` 더블클릭 → 확인 → 권한(손쉬운 사용·입력 모니터링·화면 기록·
  마이크) 리셋 + `~/.jarvis` 삭제 + `JARVIS.app`·런처 삭제. (개발 소스 `~/jarvis`는 보존)
- **윈도우**: `자비스-제거.bat` 더블클릭 → `%USERPROFILE%\.jarvis` 삭제 + 종료. 앱 폴더는 직접 삭제.

> 재빌드해도 권한이 안 섞이게 하는 근본 해결은 Apple Developer ID 코드사인(별도)이다.

## 11. 이 버전(v4.0.0) 변경 내역

**새 기능 — 에이전트 기반(Agentic Substrate) 강화**

- 🔒 **보안 게이트(느슨한 모드)** — 로컬 도구는 대부분 자동 실행되고, 외부 발송(메시지·메일)
  과 파일 삭제처럼 되돌리기 어려운 행동만 음성 확인을 요청합니다.
  `rm -rf /`, `~/.ssh` 등 자격증명 경로, `curl … | sh` 같은 파국적 명령은 확인 없이
  **항상 차단**됩니다(LLM 선의에만 의존하지 않음).

- ⚙️ **`bash_auto_allow` 토글** — 설정 파일에서 `bash_auto_allow: false`로 두거나
  환경변수 `JARVIS_BASH_AUTO_ALLOW=0`을 설정하면, 모든 셸 명령을 매번 음성으로 확인한 뒤
  실행합니다. 보안이 더 중요한 환경에서 유용합니다.

- 🧩 **플러그인** — `~/.jarvis/plugins/<이름>/` 디렉터리에 Claude Code 플러그인을 두면
  자비스가 자동으로 발견합니다. `plugins_enabled: true`(또는 `JARVIS_PLUGINS_ENABLED=1`)로
  켜야 동작합니다. 제3자 플러그인은 기본적으로 **비신뢰** 처리되며, 자동 실행이 필요하면
  `~/.jarvis/plugins/trust.json`에 `{"<이름>": true}`를 추가해 신뢰를 명시해야 합니다.
  (플러그인 자동 다운로드는 다음 버전 예정)

- 🛡️ **권한 자가복구** — 시작 시 손쉬운 사용·입력 모니터링·화면 기록·마이크 권한을 자동으로
  점검합니다. 필요한 권한이 꺼져 있으면 해당 시스템 설정 창을 열고 안내합니다.
  화면 제어·캡처 도구가 실행 중 권한에 막히면, 그 순간 바로 설정을 열어 재요청합니다
  (같은 권한에 대해서는 중복 팝업 없이 한 번만 안내).

## 12. 이전 버전(v3.0.1) 변경 내역

**새 기능**
- ⌨️ **타자로 자비스와 대화** — 전역 단축키(기본 ⌥Space / Ctrl+Space)로 떠오르는 **채팅창**에서
  타이핑으로 명령·대화. 답은 텍스트(필요하면 🔊로 음성, "음성" 토글로 자동 읽기). 명령도 타자로
  됨. 조용한 환경·긴 입력에 편리. (2장)
- 🟣 **오브 크기 단축키 토글** — 전역 단축키(기본 ⌥O)로 중앙 크게 ↔ 우측 하단 작게 즉시 전환. (5장)
- 🏷️ **이름 변경 + 별칭** — 자비스 이름을 바꿔도 STT 변형 탓에 안 깨어나던 문제 해결. 설정에서
  **별칭**(쉼표 구분)을 넣으면 그 발음들로도 호출된다.
- 🔧 단축키(타자창·오브)는 **설정 창에서 변경** 가능.

**버그 수정·정리**
- 🔇 **오류 음성 반복 차단** — 같은 오류 상황에서 "오류가 발생했습니다"를 끝없이 반복하던 것 수정
  (쿨다운·중복 차단 — 시각 알림은 유지).
- 🗑️ 의미 없어진 **"개인용 풀음성 업그레이드" 버튼 제거**(Pocket이 이미 앱에 내장됨).
- 🛡️ 타자/음성/원격 동시 사용 시 두뇌 충돌(응답 훔치기) 방지, 🔊 재생 후 상태가 멈추던 문제,
  타자 입력창 유령 창 방지 등 수정.

## 13. 이전 버전(v3.0.0) 변경 내역

**새 기능**
- 🎙️ **Pocket 무설치 번들(맥·윈도우)** — torch·음색 가중치를 앱에 내장. 받으면 바로 진짜 자비스
  영어 음색으로 말한다(별도 설치·HF 토큰 불필요). Pocket 실패 시 **edge → say 3중 폴백**.
- 🪟 **윈도우 지원 강화** — Pocket 번들 + edge 음성 폴백(System.Speech).
- 🗑️ **제거기** 동봉(맥/윈도우, 10장).

**안정성·버그 수정(코드 감사 4라운드, 66건)** — 주요만:
- 음성: 배포 앱 무음(aiosignal 번들 누락)·Pocket 폴백·바지인 후 무음·TTS 워커 데드락·
  self-echo(자비스가 제 음성을 명령으로 받던 것) 수정.
- STT: 윈도우 음성인식 영구 무반응(모델 repo 불일치) 수정.
- 도구: osascript/pmset/screencapture/cliclick 등 **타임아웃 누락으로 이벤트 루프 동결** 다수 수정,
  AppleScript 이스케이프·확인 게이트·크로스플랫폼 분기 보강.
- 두뇌: GPT(구독) 토큰 만료 시 영구 401 복구, Gemini/GPT 언어 설정 반영.
- 부팅: 손상된 setup/기억/usage 파일이 앱을 못 켜게 하던 것 방지.
- 능동알림: 동시 만기 타이머·작업 완료 알림이 소실되던 것 수정.
