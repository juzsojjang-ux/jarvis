# HUD · 오브 재디자인 설계

작성: 2026-06-14 · 상태: 설계 확정(구현 대기)

## 목표

자비스 HUD 전체(오브·패널·자막·구도·등장 애니메이션)를 영화 자비스(어벤져스2
골든 오브 + 토니 작업실 시안 홀로그램)에 가깝게 재설계한다. "픽셀 단위 영화 동일"은
절차적으로 불가능하므로, **영화를 정교하게 재현한 실사급 오브 영상 자산**을 비디오
텍스처로 쓰고, 그 위에 자비스 상태·음성에 반응하는 연출과 시안 패널을 얹는다.

## 핵심 결정 (브레인스토밍 합의)

1. **범위**: HUD 전체 재설계.
2. **구도 — A↔B 전환**:
   - **A (평소/idle)**: 화면 우하단에 작은 오브. 은은하고 방해 최소.
   - **B (활성/말하기·생각·패널표시)**: 오브가 화면 중앙으로 커지며 피어오르고,
     시안 홀로그램 패널이 주위에 펼쳐짐(토니 작업실).
   - 자비스 상태(idle→listening/thinking/speaking)로 A↔B 전환. 끝나면 A로 수축.
3. **오브 = 비디오 텍스처** (절차적 3D 아님):
   - 자산: 사용자 제공 ArtStation 팬제작 골든 네트워크 구 영상
     (1600×1600, 6초, 30fps, 검정 배경). **개인용·비공개 한정**(팬아트 — 폴 베타니
     음성 복제와 같은 전제, 배포·판매 절대 불가).
   - **검정 제거**: 빛-on-검정이므로 화면에 `mix-blend-mode: screen`(또는 캔버스
     가산 합성)으로 얹으면 검정이 빠지고 빛나는 오브만 뜬다.
4. **모션 = 회전 아님, 음성 반응**:
   - 영화 자비스는 말할 때 빙글빙글 도는 게 아니라 **음성에 맞춰 맥동/일렁임**.
   - idle: 차분(스핀 매우 느리게, 어둡게).
   - speaking: **자비스 실제 음성 크기(`level`, 이미 SSE로 HUD에 흐름)** 로
     ① 스케일 펄스 ② 골드 글로우 헤일로 ③ 코어 플래시 ④ 밝기를 구동.
   - **밝기 맥동은 은은하게**(사용자 피드백: 너무 세지 않게). 스케일·글로우가 주연.
5. **패널 = 시안 작업실 홀로그램**: 반투명, 모서리 컷, 시안 글로우, 코너 브래킷,
   무작위 폭 룰라인, 헤더(상태점+제목). B에서 오브 주위에 배치.
6. **색 정체성**: 골드 오브 + 시안 패널(영화 색 구성 그대로).
7. **공통**: `orb.html` 한 파일을 맥·윈도우가 공유. 영상 자산은 번들.

## 구성 요소 (단위)

- **오브 레이어**: `<video>`(루프, muted, autoplay) + `mix-blend-mode: screen`.
  JS가 SSE `level`/`state`로 CSS `transform: scale()`·`filter: brightness()`와
  글로우 div의 opacity를 갱신. A↔B는 컨테이너의 transform(위치·크기) 전환.
- **글로우/코어 레이어**: 오브 뒤 골드 radial-gradient div, opacity가 level에 반응.
- **패널 시스템**: 기존 notice/패널을 시안 홀로그램으로 재스타일 + B 배치 슬롯.
- **자막**: 기존 `[KO]` 자막(하단 넷플릭스풍 박스) 유지.
- **HUD 프레임**: 코너 브래킷, J.A.R.V.I.S 헤더, 옅은 그리드 — B에서만 진하게.
- **상태 머신 연동**: orb_server가 보내는 `{state, level, text, notice}` 그대로
  사용(새 채널 불필요). state로 A↔B, level로 반응.

## 자산 처리

- 8.3MB(1600²) → 번들용으로 **720×720·H.264·6초 루프**로 다운스케일(~1–2MB)해
  `voice_models/` 또는 `jarvis/hud/assets/`에 두고 orb_server가 서빙.
- 루프 이음새: 6초 영상이 매끄럽게 도는지 확인(안 매끄러우면 크로스페이드 루프).

## 반응 파라미터 (튜닝 가능, 초기값)

- 스케일: `1 + 0.22*level` (B기준) · 글로우 opacity: `0.15 + 0.5*level`
- 코어 플래시: 반경 `0.05~0.12*level` · **밝기: `1 + 0.10*level`**(은은하게 — 약화)
- idle 스핀: 재생속도 ~0.4x. speaking 전환: 0.25s ease.

## 비목표(YAGNI)

- 절차적 3D 오브(WebGL) — 영상 자산으로 대체하므로 불필요.
- 영화 프레임 픽셀 단위 복제 — 원리상 불가, 추구 안 함.
- 다중 영상 자산/테마 전환 — 1개 오브로 시작.

## 검증

- 맥에서 자비스 띄워 실제 투명 오버레이로 A↔B·음성 반응 육안 확인.
- 윈도우는 pywebview 투명 오버레이에서 동일 확인(실기 필요).
- 단위 테스트: orb_server 메시지 계약(state/level/notice) 회귀, 자산 경로 존재.
