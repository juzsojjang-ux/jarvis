## 추가 인사이트 (라운드2)

- **폰트**: 영화 HUD = `0Arame / 0Arame Mono`(Hitype, Jayse Hansen). 유료·IP라 번들 불가 →
  가까운 무료 지오메트릭 모노(예: 시스템 mono / Oxanium·Chakra Petch류 무료 대체)로 근사.
- **Omega 위젯**(Mark III): 키홀형이 열리며 **부채꼴 펼침**으로 하위기능 노출 = 우리 A↔B(코너→중앙 확장)와 동일.
- **모션 원칙(원작 팀)**: iPhone식 **단순 zoom/wipe** + 모션블러로 매끄럽게. 과한 연출 금지(우리 "빙글빙글 금지"와 일치). 0.3~0.4s ease.
- **Kent Seki 3원칙**: ① 퍼포먼스(=내용)가 그래픽 배치를 정한다 ② Z-깊이(보이지 않는 뒤 공간도 설계) ③ 매 샷 "alpha event"(핵심 한 가지). + "기능 없으면 뺀다".
- **색 진화**: Mark II 시안 → Mark III는 화이트+액센트("cyan is passé"). 우리는 어벤져스기 **골드 오브 + 시안 패널** 정체성 유지(우리 색 결정).
- **글라스 홀로그램 질감**(IM3): 2D 그래픽이 아니라 빛 상호작용으로 유리처럼. 레이저 발광원 모티프 + 베이스 선반(Z-깊이).
- **원작 영감**: 실제 항공 HUD + 2001 스페이스 오디세이(헬멧 반사). "HUD는 캐릭터(JARVIS)다."

**색**
- 시안 프라이머리 ≈ **#67C7EB**(글로우 강). 보조 텍스트 #aef/#d8f6ff.
- 골드/앰버 액센트 **#FBCA03 / #B97D10 / #ECBA4F** — 경고·강조·오브.
- 베이스: 거의 검정(투명 위), 패널 바탕 rgba(0,8,14,~.85). 청색 와이어프레임 모션그래픽.

**기하·모티프**
- 원형 레티클/**동심원 링**(깊이에서 확장 진입), **점선 호 게이지**(느린 회전, 부분 채움).
- **코너 브래킷** ⌐(패널·화면 모서리), 육각/사다리꼴 + 굵은 시안 글로우 테두리.
- **리더 라인/테더** + 끝점 닷 + 라벨 콜아웃, 끊임없는 **데이터 틱 스트림**, 옅은 그리드.
- 와이어프레임 3D 스키매틱(슈트/부품), **스테레오 깊이 레이어**(C4D 렌더 3D 느낌).

**타이포**: 응축형 기술 산세리프 + 모노 수치 readout, 대문자·넓은 자간, 작은 라벨.

**핵심 행동 원리 (우리 설계와 직결)**
- **주변부 게이지는 작게 대기 → 필요할 때 중앙으로 커진다** = 우리 **코너(A)↔중앙(B)** 와 정확히 동일.
- HUD 하단 **5개 상시 아이콘**: suit status / targeting·optics / radar / artificial horizon / map → 우리 텔레메트리 패널의 모티프(시계·NET·MIC·CPU/MEM·작업수).
- **주의 에스컬레이션**: 저우선 데이터는 숨기고 문제를 메인 화면에 키워 보여줌 → notice 우선순위 로직.
- **모든 요소는 기능을 가진다(가짜 장식 최소)** → 텔레메트리·게이지는 진짜 데이터에 바인딩.
- 작업실: 홀로그램을 손으로 스와이프/회전, **폭발 분해(exploded)** 블루프린트, 숨은 구조 드러내기.
- 라디얼 "진단 위젯" = 정보량(altitude)에 따라 펼침/접힘.
