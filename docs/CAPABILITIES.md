# 자비스 능력 목록 (2026-06-11 기준, 도구 28개 + 풀 도구 개방)

말 걸기: **"자비스"** 라고 부르면 됨(웨이크워드). 답변 후 8초는 웨이크워드 없이
연속 대화. 백업: 오른쪽 옵션 키 누른 채 말하기(PTT). 말 끊기(바지인) 가능.

## 1. 묻고 알아보기

| 말 | 동작 |
|---|---|
| "지금 몇 시야?" | 현재 날짜·시간 |
| "서울 날씨 어때?" | 실시간 날씨(서울·부산·인천·대구·대전·광주·수원·제주) |
| "배터리 얼마나 남았어?" | 잔량+충전 상태 |
| "오늘 일정 뭐야?" / "미리알림 보여줘" | 캘린더·미리알림 읽기 |
| "최근 메시지 읽어줘" / "안 읽은 메일 알려줘" | 메시지·메일 읽기(읽기 전용) |
| "어제 F1 결과 알려줘" 등 최신 정보 | 웹 검색해서 답변 |
| "바탕화면에서 보고서 파일 찾아줘" / "그 파일 읽어줘" | 파일 검색·읽기(Glob/Grep/Read) |

## 2. 맥 조작

| 말 | 동작 |
|---|---|
| "사파리 열어줘" / "음악 앱 꺼줘" | 앱 열기/종료 |
| "볼륨 30으로" / "음소거 해줘" | 볼륨·음소거 |
| "다크 모드 켜줘" / "와이파이 꺼줘" / "블루투스 켜줘" | 시스템 토글 |
| "밝기 올려줘" / "화면 꺼줘" / "잠자기" | 밝기·절전 |
| "화면 잠가줘" | 화면 잠금 |
| "클립보드 읽어줘" / "이거 클립보드에 복사해줘" | 클립보드 읽기/쓰기 |
| "OO 단축어 실행해줘" / "단축어 목록 알려줘" | macOS 단축어(스마트홈·방해금지 등 무한 확장 훅) |
| 그 밖의 맥 조작 | control_mac(범용 AppleScript)로 두뇌가 즉석 처리 |

## 3. 음악·미디어

| 말 | 동작 |
|---|---|
| "아이유 노래 틀어줘" | 라이브러리에서 곡/아티스트/플레이리스트 검색 재생 |
| "멈춰" / "다음 곡" / "이전 곡" / "지금 무슨 노래야?" | 재생 제어 |

## 4. 화면 보기·조작 (3c)

| 말 | 동작 |
|---|---|
| "지금 화면에 뭐 있어?" | 화면 캡처 → 두뇌가 이미지를 직접 보고 설명 |
| "화면 제어 모드 켜줘" | 화면 조작 게이트 열기(5분 자동 만료, "꺼줘"로 즉시 종료) |
| (모드 중) "메모 앱 열어서 안녕이라고 적어줘" | 캡처→좌표 파악→클릭·입력·스크롤 반복 수행 |

화면 캡처(보기)는 모드와 무관하게 항상 가능. 조작은 모드가 켜져 있을 때만.

## 5. 타이머·미리알림·메모

| 말 | 동작 |
|---|---|
| "라면 타이머 3분 맞춰줘" | 타이머(완료 시 음성 알림, 여러 개 동시) |
| "타이머 취소" / "타이머 얼마 남았어?" | 취소·목록 |
| "내일 9시에 약 먹기 미리알림 추가해줘" | Reminders 추가 |
| "메모에 OO 적어줘" | Notes 새 메모 |

## 6. 통역 모드 (3b)

"통역 모드 켜줘" → 한국어로 말하면 **영어로**, 영어로 말하면 **한국어로** 발화.
연속 통역(follow-up 유지). "통역 모드 꺼줘"로 복귀. 켜는 순간 백그라운드
예열로 첫 통역부터 빠름.

## 7. 기억

| 말 | 동작 |
|---|---|
| "나 매운 거 못 먹는 거 기억해" | 장기 기억 저장 |
| (명시 없이도) 대화 중 선호·약속·습관 | 자동으로 조용히 저장(중복 방지) |

## 8. 일·코딩 시키기 (풀 도구 개방)

두뇌는 Claude Code 풀 도구(bash, 파일 읽기/쓰기/편집, 검색)를 가짐.
"바탕화면 스크린샷 전부 폴더로 정리해줘", "이 폴더에 README 만들어줘" 같은
실제 작업 수행. **파괴적 단계(Bash/파일 쓰기·편집)는 실행 전 음성으로
"~할까요?" 확인** — "네"라고 해야 실행("아니"·무응답=거부).

## 9. 능동적 자비스 (먼저 말 걸기)

- **아침 브리핑**: 그날 첫 화면 잠금 해제 시 날씨+미리알림+일정 요약
- **배터리 경고**: 20/10/5% 하향 돌파 시 1회씩, 충전 시작·완충도
- **일정·미리알림 사전 알림**: 시작 10분 전
- **복귀 인사**: 오랜만에 자리로 돌아오면(쿨다운 4시간)
- **타이머 완료**: "주인님, 라면 타이머가 끝났습니다"

대화 중에는 끼어들지 않고 한가할 때만 말함.

## 10. 말투·표시

- 영어 자비스 음색(Pocket TTS, 폴 베타니 클로닝) + **화면에 한국어 자막**
- 매 답변 자비스식 건조한 위트, "sir/주인님" 호칭
- 영화풍 원형 HUD 오버레이(음성에 반응해 움직임)
- "깊게 생각해봐/최대 사고" 붙이면 그 턴만 확장 사고(느리지만 똑똑)
- 즉답 필러("One moment, sir.")로 대기 시간 체감 제거
- 매 턴 지연 로그 `[지연] STT·두뇌 첫문장`(터미널, 튜닝용)

## 안전 장치(요약)

읽기는 자유, **쓰기·실행은 음성 확인**, **화면 조작은 모드 게이트+자동 만료**,
오인식 한 마디로는 어떤 파괴적 동작도 불가. 비자비스 발화는 즉시 폐기(로그 없음).
